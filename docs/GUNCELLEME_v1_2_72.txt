Kurulum:
1. ZIP'i site/proje kök dizinine yükleyip mevcut dosyaların üzerine yazdırın.
2. Admin açıkken /update/updater.php dosyasını bir kez çalıştırın.
3. Projeyi GitHub reposuna yükleyin.
4. GitHub → Actions → Build Android APK and AAB → Run workflow.
5. Artifacts bölümünden release AAB dosyasını indirin.
