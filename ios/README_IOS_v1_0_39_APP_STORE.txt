Akıllı Zikir & Hatim v1.0.39 - iOS / App Store Hazırlığı

Bu klasör geliştirici/admin içindir. Mobil kullanıcıya görünmez.

Gerekenler:
- Apple Developer Program hesabı
- Mac bilgisayar veya macOS build runner
- Xcode
- App Store Connect uygulama kaydı
- Bundle ID: com.ilhanbeluk.akillizikirhatim
- TestFlight test süreci
- Gizlilik politikası URL
- App Store açıklama ve ekran görüntüleri

Önerilen süreç:
1. Admin panel > iOS Build Merkezi ayarlarını doldur.
2. Apple Developer hesabında Bundle ID oluştur.
3. App Store Connect'te uygulama kaydı aç.
4. Capacitor iOS projesi oluştur.
5. TestFlight build üret.
6. iPhone üzerinde test et.
7. Gizlilik ve App Store bilgilerini tamamla.
8. App Review'a gönder.
